使用済み切手に強い！

当店にお越しのお客様の95%は使用済み切手のコレクターさんです。 それゆえ品揃えも国内最大級！買取も販売も使用済み切手に関して右に出る店はありません！

取り扱い使用済み切手

  • ✓ 手彫～現行まで50冊以上のファイル
  • ✓ 記念、シリーズもの、ふるさと切手
  • ✓ 消印物・満月印コレクション
  • ✓ エンタイア（封筒・はがき付き）

外国切手に強い！

外国切手はいわば切手界の専門領域。買取の査定に差が出るのはこのような専門知識が必要な分野です。 少しでも正しい査定で高く売るなら当店へお越しください。

外国切手の強み

  • ✓ 各国貼り込み帳700冊以上の在庫
  • ✓ 専門知識による適正な査定価格
  • ✓ 中国切手の高価買取実績多数

広島のショップにはあちこちから切手の蒐集家が集まります。 好きなものに囲まれ、楽しさを分かち合えるコレクターに出会えるのも魅力の一つ。

配送・送料について

郵便局（書留便など）

郵便局の書留便、ゆうパックなどでお送りします。お客様の玄関先まで確実にお届けします。

料金は全国一律650円です。

郵便局（特定記録便など）

郵便局の特定記録便などでお送りします。基本的にお客様の郵便受箱に配送となります。

料金は全国一律400円です。

※特定記録便は万が一、亡失、誤配、商品破損、盗難などがあった場合に補償を受ける事ができません。
※平日のみの配達となります。土日、祝日は配達がありません。

店頭お受け取り

ご注文品の用意が整いましたら、メールでお知らせいたしますので、メール受信後に店舗（広島市中区）にてお受け取りください。
